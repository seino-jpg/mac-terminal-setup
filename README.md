# Mac Terminal Setup

Claude Code と使うためのターミナル設定。タブ名にセッションの日本語タイトルをシンプルに表示する。

## セットアップ

### 1. Homebrew プロファイルをインポート

`Homebrew.terminal` をダブルクリック → ターミナルにプロファイルが追加される。

その後、ターミナル → 設定 → プロファイル → Homebrew → 「デフォルト」ボタンをクリック。

### 2. タイトル設定（ウインドウ・タブ タブ）

以下のチェックボックスを **すべてオフ** にする（ウインドウ・タブ 両方）：

| 項目 | 設定 |
|---|---|
| 作業ディレクトリまたは書類 | オフ |
| 動作中のプロセス名 | オフ |
| 引数 | オフ |
| 大きさ | オフ |

これにより、Claude が `printf '\033]0;🤖 <トピック>\007'` でセットした日本語タイトルだけがタブに表示される。

- ActivityIndicator（アクティビティ表示）はオンのまま推奨（処理中かどうかが一目でわかる）

### 3. Claude Code 側の設定

`~/.claude/CLAUDE.md` または `~/.claude-business/CLAUDE.md` に以下を追加：

```markdown
# Terminalタイトル設定ルール

会話の内容が把握できた時点で、必ず以下のBashコマンドでTerminalタイトルをセットすること。

\`\`\`bash
printf '\033]0;🤖 <トピック>\007'
\`\`\`

- タイトルは20文字以内で簡潔に（例: `🤖 gas-kpi-cs 解約分析`, `🤖 Slack通知 設定`）
- 会話の最初のメッセージを読んだ直後にセットする
- 作業内容が途中で変わった場合は更新する
```

## 仕組み

`printf '\033]0;TITLE\007'` はターミナルのエスケープシーケンスでウインドウ/タブタイトルを書き換える。Mac Terminal はこの値をそのままタブ名として表示する（他の情報をオフにしている場合）。
